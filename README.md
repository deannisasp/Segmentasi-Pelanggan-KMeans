# Segmentasi-Pekanggan-KMeans
Proyek ini berfokus pada analisis segmentasi pelanggan menggunakan metode KMeans clustering untuk mengelompokkan pelanggan berdasarkan karakteristik dan performa event. Tujuannya adalah menghasilkan insight berbasis data yang dapat mendukung pengambilan keputusan bisnis dan optimalisasi strategi pemasaran.
