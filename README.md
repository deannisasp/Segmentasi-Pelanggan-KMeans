# ğŸ› ï¸ Methodology
Tahapan analisis segmentasi pelanggan dilakukan secara sistematis sebagai berikut:
1. Data Collection: Mengumpulkan data laporan event dari aktivitas marketing Indosat.
2. Data Cleaning & Preprocessing: Pengecekan missing value, Penanganan outlier, Normalisasi data numerik, Exploratory Data Analysis (EDA), Analisis distribusi data, Visualisasi boxplot sebelum dan sesudah penanganan outlier
3. Clustering Analysis: Menentukan jumlah cluster optimal menggunakan Elbow Method, Membangun model clustering menggunakan K-Means, Evaluasi hasil clustering menggunakan Silhouette Score
4. Interpretation & Visualization: Visualisasi hasil clustering, Analisis karakteristik tiap cluster, Pemetaan cluster berdasarkan wilayah dan kategori event
